# 🫀 Systole
**Systole** is a sleek, modern, and immersive smart home controller for Philips Hue lighting systems. Built with SwiftUI and modern Swift concurrency, it offers a high-performance "Glassmorphic" interface for managing your home's atmosphere with precision.

![Swift](https://img.shields.io/badge/Swift-5.9-orange.svg)
![Platform](https://img.shields.io/badge/Platform-iOS%2017%2B-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

---

## ✨ Features

- **Dynamic Dashboard**: Real-time overview of all connected Hue lights with state-aware iconography.
- **Immersive Control**: A detail view featuring high-fidelity color gradients, brightness sliders, and power toggles.
- **Atmospheric Presets**: One-tap access to lighting "moods" (Warm, Cool, Vibrant colors).
- **Custom Color Engine**: Fine-grain color selection using a native SwiftUI color wheel that translates directly to Hue HSB values.
- **Contextual Awareness**: Integrated weather and location services to display your local environment alongside your home controls.
- **Squish UI**: Custom interaction physics for a tactile, responsive feel when navigating.
